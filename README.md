# Задачник JavaScript по уровням

Решение задач  по JS по уровням с [сайта](https://code.mu/ru/javascript/tasker/stager/).

Кол-во решённых задач на данный момент: **52**. Завершён уровень 2.5.

## Запуск тестов
```bash
pnpm install --frozen-lockfile

pnpm test
```

## Структура проекта

```text
src
  └─ [ level_* ]
          ├─ level_*_*.ts
          └─ level_*_*.spec.ts
```
